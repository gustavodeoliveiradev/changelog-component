# 📋 Changelog Component

> Um componente de changelog moderno, responsivo e acessível construído com HTML5 semântico e CSS3 modular.

## 🚧 Em Desenvolvimento

Este projeto está sendo construído passo a passo. Acompanhe o progresso:

- [x] **Dia 1** — Estrutura HTML semântica + Setup modular
- [ ] **Dia 2** — Reset CSS + Variáveis CSS + Tipografia base
- [ ] **Dia 3** — Layout principal (Flexbox/Grid) + Timeline vertical
- [ ] **Dia 4** — Componentes: Cards de versão, badges, datas, tags
- [ ] **Dia 5** — Responsividade + Animações + Polimento final

## 🎯 Objetivos

- [ ] HTML semântico e acessível (ARIA)
- [ ] CSS modular com variáveis (Design Tokens)
- [ ] Timeline vertical com linha conectando releases
- [ ] Cards de release com categorias (feature, improvement, bugfix)
- [ ] Badges coloridos por tipo de mudança
- [ ] Animações suaves ao scrollar
- [ ] Modo escuro/claro (dark/light mode)
- [ ] 100% responsivo (mobile-first)

## 🛠️ Tecnologias

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- JavaScript (Vanilla)
